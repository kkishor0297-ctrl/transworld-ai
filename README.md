# Transworld AI - Language Barrier Solver 🌐🤖

**एक छोटी सी कोशिश, AI की मदद से।** यह ऐप मैंने अपने मोबाइल और लैपटॉप से बनाया है ताकि दुनिया भर के यात्रियों (Travelers) और इंटरव्यू देने वाले लोगों की मदद की जा सके।

## 🎯 मुख्य उद्देश्य (Goal)
इंटरव्यू और यात्रा के दौरान आने वाली भाषा की बाधा (Barrier) को खत्म करना।

## 🌍 समर्थित भाषाएँ (Top 20 World Languages)
दुनिया में सबसे ज़्यादा बोली जाने वाली टॉप 20 भाषाएँ शामिल की गई हैं (बोलने वालों की संख्या के क्रम में):

English, Mandarin (Chinese), Hindi, Spanish, French, Arabic, Bengali, Portuguese, Russian, Urdu, Indonesian, German, Japanese, Swahili, Marathi, Telugu, Turkish, Tamil, Vietnamese, Korean.

## 🎙️ मुख्य फीचर्स (Features)

- 🎤 **Voice Input:** माइक बटन दबाकर बोलें, ऐप अपने आप आपकी भाषा को टेक्स्ट में बदल देगा।
- 🔄 **Language Select:** "Speak in" और "Translate to" — दोनों भाषाएँ अपनी पसंद से चुनें।
- ⇄ **Swap:** एक टैप में दोनों भाषाएँ बदलें।
- 📝 **Text Translation:** टाइप करके भी ट्रांसलेट कर सकते हैं।
- 🔊 **Voice Output:** ट्रांसलेशन को सुनें — चुनी हुई भाषा में आवाज़ में जवाब मिलेगा।
- ⧉ **Copy:** ट्रांसलेशन को एक टैप में कॉपी करें।

## 📱 Mobile + Laptop दोनों से बना Project
यह प्रोजेक्ट मोबाइल पर शुरू हुआ और अब लैपटॉप पर पूरा किया गया है। यह साबित करता है कि अगर इच्छाशक्ति हो, तो तकनीक की मदद से हम घर बैठे बड़े समाधान बना सकते हैं।

## ⚙️ Tech Stack
- HTML, CSS, JavaScript (Vanilla)
- Web Speech API (Voice input/output)
- MyMemory Translation API (Translation)
- Progressive Web App (PWA) — installable, works offline (app shell)
